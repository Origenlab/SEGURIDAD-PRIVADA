## REQUISITOS OBLIGATORIOS

### Contenido
- Extension: 1500-2200 palabras (contenido narrativo, sin contar HTML)
- 100% original, sin plagio, sin contenido generico de IA
- Informacion verificable y util para el lector
- Enfocado en resolver dudas reales del cliente

### SEO On-Page
- Keyword principal: densidad 1-1.5%
- Incluir minimo 12 keywords LSI relacionadas
- Usar sinonimos y variaciones semanticas naturales
- Cada H2 debe contener variacion de keyword
- Meta description: 150-160 caracteres con CTA implicito

### Tono y Estilo
- Profesional pero cercano (no corporativo frio)
- Experto que asesora, no vendedor que presiona
- Usar "tu" para dirigirse al lector
- Evitar superlativos exagerados ("el mejor", "increible", "fantastico")
- Evitar frases genericas de IA ("en el mundo actual", "es importante destacar", "cabe mencionar")
- Transiciones naturales entre secciones
- Parrafos de 3-5 lineas maximo

### Prohibiciones Estrictas
- NO mencionar precios especificos
- NO inventar estadisticas o datos falsos
- NO usar contenido copiado de otras fuentes
- NO repetir frases o estructuras
- NO usar emojis
- NO mencionar competidores
- NO hacer promesas exageradas

## Notas de Uso

### Para N8N/Make/Zapier:

1. Copia todo el bloque de texto entre las comillas triples
2. Reemplaza `[INSERTAR TEMA AQUI]` con el tema especifico
3. Envia al modelo de IA (GPT-4, Claude, etc.)
4. Parsea el JSON de respuesta
5. Usa `contenidoHTML` para la variable `{{CONTENIDO}}`
6. Genera `{{FAQ_SCHEMA}}` a partir de `faqSchema`

### Variables que genera:

| Variable | Campo JSON |
|----------|------------|
| `{{TITULO}}` | `titulo` |
| `{{SLUG}}` | `slug` |
| `{{META_DESCRIPTION}}` | `metaDescription` |
| `{{META_KEYWORDS}}` | `metaKeywords` |
| `{{BREADCRUMB_TEXT}}` | `breadcrumbText` |
| `{{CATEGORIA}}` | `categoria` |
| `{{TIEMPO_LECTURA}}` | `tiempoLectura` |
| `{{CONTENIDO}}` | `contenidoHTML` |
| `{{FAQ_SCHEMA}}` | Generar de `faqSchema` |

### Generar FAQ Schema:

```javascript
function generarFAQSchema(faqArray) {
    return JSON.stringify({
        "@context": "https://schema.org",
        "@type": "FAQPage",
        "mainEntity": faqArray.map(faq => ({
            "@type": "Question",
            "name": faq.pregunta,
            "acceptedAnswer": {
                "@type": "Answer",
                "text": faq.respuesta
            }
        }))
    });
}
```
